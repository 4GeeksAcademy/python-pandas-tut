# `04.2` Series Apply

Dada una serie variable como:

```py
my_series = pd.Series([2, 4, 6, 8, 10])
```

## 📝 Instrucciones: 

1. Usa la función `my_series.apply` para dividir todos los números de la siguiente serie por 2.

## Resultado Esperado:

```bash
0    1.0
1    2.0
2    3.0
3    4.0

```
