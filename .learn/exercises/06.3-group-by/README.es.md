# `06.3` Group By

¿Cuántos nombres diferentes existen en el Dataset?

## 📝 Instrucciones:

1. Usa la función de DataFrame `groupby`, para agrupar tu tabla por nombre.

3. Usa la función `len(result)` para contar el número de grupos.
